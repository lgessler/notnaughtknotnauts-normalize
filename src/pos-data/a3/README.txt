Extracted tokens and POS tags from the English Web Treebank 
via the Universal Dependencies Project (http://universaldependencies.org/).
POS tags come in two varieties: Universal (v2), in .upos files, 
and new-style Penn Treebank tags, in .ppos files.

Tagset overviews:

  * Universal: http://people.cs.georgetown.edu/nschneid/p/UPOS-English.pdf
  * Penn (full new-style tagset): https://spacy.io/docs/usage/pos-tagging#pos-tagging-english
  * Penn (examples): http://surdeanu.info/mihai/teaching/ista555-fall13/readings/PennTreebankTagset.html

Data files:

en-ud-train.upos.tsv
en-ud-dev.upos.tsv
en-ud-test.upos.tsv
en-ud-train.ppos.tsv
en-ud-dev.ppos.tsv
